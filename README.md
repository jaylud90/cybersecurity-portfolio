# Cybersecurity Portfolio – Jay Ludington

## Overview

This portfolio demonstrates hands-on experience in web application security testing. It includes real-world vulnerability discovery, exploitation, and documentation using tools such as Burp Suite and browser developer tools.

## Skills Demonstrated

* Cross-Site Scripting (DOM, Reflected, Stored, WAF Bypass)
* Insecure Direct Object Reference (IDOR)
* Privilege Escalation (Mass Assignment)
* API Security Testing

## Tools Used

* Burp Suite
* Browser Developer Tools
* Kali Linux

## Projects

### 1. IDOR – BasketItems API

Unauthorized access to user data by modifying object identifiers.

### 2. Stored XSS

Persistent script execution affecting all users viewing stored content.

### 3. Reflected XSS

Script injection via request parameters executed in server response.

### 4. XSS Filter Bypass (WAF)

Bypassed filtering mechanisms using allowed tags and event handlers.

## Goal

To obtain an entry-level cybersecurity role and continue developing practical offensive and defensive security skills.
